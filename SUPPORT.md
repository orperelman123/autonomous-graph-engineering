# Support

Use [GitHub Discussions](https://github.com/orperelman123/autonomous-graph-engineering/discussions) for usage questions, design discussion, and sharing non-sensitive workflows.

Use the [issue chooser](https://github.com/orperelman123/autonomous-graph-engineering/issues/new/choose) for reproducible bugs, feature proposals, evaluation cases, example graphs, and platform reports.

Never include credentials, private prompts, checkpoints, provider transcripts, absolute user paths, or production targets. Report suspected vulnerabilities through [GitHub private vulnerability reporting](https://github.com/orperelman123/autonomous-graph-engineering/security/advisories/new).

This is a community-maintained open-source project. Support and response times are not guaranteed.
